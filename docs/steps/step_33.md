# Step 33: Charts UX overhaul

## Цель

Сделать вкладку `Charts` пригодной для операционного использования: единые dropdown-контролы, корректные серии, временные диапазоны и визуализация изменений users.

## Реализация

[x] Перевести controls графиков на единый паттерн dropdown-меню справа.
[x] `Requests Per Second`:
  - dropdown метрик (порядок: step -> http -> other);
  - отдельный dropdown для серии `Users`;
  - визуальные маркеры точек изменения users.
[x] Добавить переключатель диапазона `5/15/30 minutes/All time` (по умолчанию `All time`) для обоих графиков.
[x] `Latency`:
  - dropdown выбора метрик;
  - dropdown выбора вида latency (`average|median|p95|p99`).
[x] Хранить историю графиков за run и фильтровать по выбранному диапазону.

## Прогресс

- [x] UX графиков унифицирован через dropdown.
- [x] Выбор series/метрик реализован для обоих графиков.
- [x] Диапазоны времени и users-change markers работают.
